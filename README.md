# Secure User Management Backend

A Spring Boot backend for secure user management with JWT authentication.  

## Features
- User registration
- User login with JWT generation
- Fetch logged-in user profile
- Spring Security integration

## Tech Stack
- Java 21
- Spring Boot 3.1.5
- Spring Security
- JWT
- Maven
